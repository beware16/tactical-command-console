# tactical-command-console
# 🛰️ Project ORION: Tactical AI Command Interface
**Category:** Geospatial AI & Intelligence Dashboards  
**Codename:** ORION-V4  

## ⚡ Overview
Project ORION is a browser-based, high-fidelity situational awareness dashboard designed with a "Tactical Command Center" aesthetic. It simulates a CIA-style geospatial monitoring tool, integrating live satellite imagery, real-time asset tracking, and AI-driven intelligence feeds.

## 🚀 Key Features
- **🔒 Secure Uplink:** Encrypted password gate for terminal access.
- **🗺️ Geospatial Engine:** High-resolution satellite mapping powered by Mapbox GL JS.
- **✈️ Real-time Asset Monitoring:** Live simulation of ADS-B aircraft telemetry within specific AOIs (Areas of Interest).
- **🤖 Neural Intel Feed:** A scrolling tactical log simulating AI analysis of incoming signals and satellite imagery.
- **📟 CRT Visual Overlay:** Custom CSS-engineered scanlines and flicker effects for an authentic terminal experience.
- **📡 Multi-Feed Surveillance:** Integrated visual monitoring of high-traffic transit points.

## 🛠️ Tech Stack
- **Framework:** [Next.js 14+](https://nextjs.org/) (React)
- **Mapping:** [Mapbox GL JS](https://www.mapbox.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Deployment:** [Vercel](https://vercel.com/)
